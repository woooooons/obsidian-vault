## **1. 산(Acid)과 염기(Base)의 정의**

### **(1) 아레니우스 정의 (Arrhenius, 1884)**

- **산**: 물에 녹아 H⁺(수소 이온) 내놓는 물질
    - HCl → H⁺ + Cl⁻
    - H₂SO₄ → 2H⁺ + SO₄²⁻
- **염기**: 물에 녹아 OH⁻(수산화 이온) 내놓는 물질
    - NaOH → Na⁺ + OH⁻
    - Ca(OH)₂ → Ca²⁺ + 2OH⁻
- **중화 반응**: H⁺ + OH⁻ → H₂O
- **한계**: 물에서만 적용, NH₃ 같은 염기 설명 못함

---

### **(2) 브뢴스테드-로우리 정의 (1923) ⭐ 가장 널리 사용**

- **산(양성자 주개, Proton Donor)**: H⁺를 내놓는 물질
- **염기(양성자 받개, Proton Acceptor)**: H⁺를 받는 물질

#### **예시 1: HCl + H₂O**

```
HCl + H₂O → H₃O⁺ + Cl⁻
산    염기    짝산   짝염기
```

- HCl: H⁺ 제공 → 산
- H₂O: H⁺ 받음 → 염기
- H₃O⁺: HCl의 짝염기가 아니라 H₂O의 **짝산**
- Cl⁻: HCl의 **짝염기**

#### **예시 2: NH₃ + H₂O**

```
NH₃ + H₂O ⇌ NH₄⁺ + OH⁻
염기   산     짝산   짝염기
```

- NH₃: H⁺ 받음 → 염기
- H₂O: H⁺ 제공 → 산 (양쪽성, amphoteric)
- NH₄⁺: NH₃의 짝산
- OH⁻: H₂O의 짝염기

#### **짝산-짝염기 쌍 (Conjugate Pair)**

- 산이 H⁺를 잃으면 → 짝염기
- 염기가 H⁺를 받으면 → 짝산
- **관계**: 산 ⇌ 짝염기 + H⁺

|산|짝염기|염기|짝산|
|---|---|---|---|
|HCl|Cl⁻|NH₃|NH₄⁺|
|H₂SO₄|HSO₄⁻|H₂O|H₃O⁺|
|CH₃COOH|CH₃COO⁻|HCO₃⁻|H₂CO₃|
|H₃O⁺|H₂O|OH⁻|H₂O|

### **(3) 루이스 정의 (Lewis, 1923) - 가장 포괄적**

- **산(전자쌍 받개, Electron Pair Acceptor)**: 전자쌍을 받음
- **염기(전자쌍 주개, Electron Pair Donor)**: 전자쌍을 제공

#### **예시 1: BF₃ + NH₃**

```
    F       H
    |       |
F—B  +  :N—H  →  F—B—N—H
    |       |        | |
    F       H        F H
  루이스산  루이스염기   착화합물
```

- BF₃: 전자쌍 받음 (비어있는 오비탈)
- NH₃: 전자쌍 제공 (고립 전자쌍)

#### **예시 2: H⁺ + H₂O**

```
H⁺ + :O—H  →  H—O⁺—H
루이스산 루이스염기    H
```

#### **장점**

- H⁺ 없는 반응도 설명 가능
- 금속 착화합물 설명
- 촉매 반응 이해
