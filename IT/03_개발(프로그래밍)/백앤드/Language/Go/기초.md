## 변수와 상수

### 변수

1.  **기본 선언**
  ```go
    var a int
    ```
*   `var` 키워드 + 변수명 + 타입

2.  **선언과 동시에 초기값 할당**
  ```go
    var f float32 = 11.0
    ```

3.  **값 재할당 가능**
  ```go
    a = 10
    f = 12.0
    ```

4.  **사용되지 않는 변수 → 에러 발생**
    *   반드시 사용하거나 삭제해야 함.

5.  **여러 변수 한 번에 선언**
  ```go
    var i, j, k int
    var i, j, k int = 1, 2, 3
    ```

6.  **타입 추론 가능**
  ```go
    var i = 1      // int로 추론
    var s = "Hi"   // string으로 추론
    ```

7.  **짧은 선언 (함수 내부에서만 가능)**
  ```go
    i := 1
    ```
 *   `var` 생략 가능, 함수 밖에서는 사용 불가.

### 상수

1.  **기본 선언**
  ```go
    const c int = 10
    const s string = "Hi"
    ```

2.  **타입 생략 가능 (자동 추론)**
  ```go
    const c = 10
    const s = "Hi"
    ```

3.  **여러 상수 묶어서 선언**
  ```go
    const (
        Visa   = "Visa"
        Master = "MasterCard"
        Amex   = "American Express"
    )
    ```

4.  **순차 값 자동 할당 (iota 사용)**
  ```go
    const (
        Apple  = iota // 0
        Grape         // 1
        Orange        // 2
    )
    ```

***

핵심 포인트:

*   `const` 키워드 사용
*   타입 생략 가능 (Go가 추론)
*   여러 상수는 괄호로 묶어 선언 가능
*   `iota`로 자동 증가 값 부여 가능
