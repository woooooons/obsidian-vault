### 1. 개발 및 프로그래밍 (Development)

- **Git / GitHub Desktop**: 버전 관리를 위한 표준 도구.
    
- **Node.js / Go / Python-manager**: 각 언어별 런타임 및 환경 관리자.
    
- **Qt Online Installer**: GUI 프로그램 개발을 위한 프레임워크 설치 도구.
    
- **Postman**: API 개발 및 테스트를 위한 필수적인 HTTP 클라이언트.
    
- **Obsidian**: 마크다운 기반의 지식 관리 도구로, 개발 노트를 작성하기에 최적.
    

### 2. 네트워크 및 원격 관리 (Network & Remote)

- **PuTTY / MobaXterm**: SSH, Telnet 등 원격 서버 접속을 위한 터미널 에뮬레이터. 특히 MobaXterm은 다양한 기능을 하나로 제공하여 편리.
    
- **WinSCP**: SFTP/FTP 클라이언트로, 서버와 로컬 간의 안전한 파일 전송.
    
- **Wireshark / Npcap**: 네트워크 패킷 분석을 위한 도구.
    
- **UltraVNC**: 원격 데스크톱 제어를 위한 오픈소스 소프트웨어.
    
- **IP-mate**: 효율적인 IP 주소 관리 및 네트워크 설정을 도와주는 유틸리티.
    

### 3. 데이터베이스 관리 (Database)

- **HeidiSQL**: MySQL, MariaDB, PostgreSQL 등을 지원하는 가벼운 DB 클라이언트.
    
- **Redis (x64)**: 인메모리 데이터 구조 저장소로, 로컬 테스트 환경 구축 시 유용.
    

### 4. 시스템 유틸리티 (System Utilities)

- **Everything**: Windows 내 파일을 0.1초 만에 찾아주는 강력한 검색 도구.
    
- **NSSM (Non-Sucking Service Manager)**: 일반 실행 파일을 Windows 서비스로 등록해 주는 유용한 도구.
    
- **Rufus**: 부팅 가능한 USB 드라이브를 만드는 데 사용.
    
- **Malware Zero (mzero)**: 강력한 보조 악성코드 제거 스크립트.
    
- **WSL Update**: Windows Subsystem for Linux 환경 유지를 위한 업데이트 패키지.
    

### 5. 생산성 및 그래픽 (Productivity & Graphics)

- **PicPick**: 스크린샷 캡처부터 간단한 편집, 색상 추출까지 가능한 올인원 도구.
    
- **Blender**: 3D 모델링 및 렌더링을 위한 오픈소스 소프트웨어.
    
- **VLC Media Player**: 거의 모든 포맷을 지원하는 가벼운 미디어 플레이어.
    
- **Chrome: 설명이 필요 없는 브라우저.
    
- **Hancom Office (HOfficeDN)**: 문서 업무를 위한 한컴 오피스.
