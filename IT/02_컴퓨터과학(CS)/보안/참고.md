### 명령줄 도구

**OpenSSL**

```bash
# 설치 확인
openssl version

# 파일 암호화
openssl enc -aes-256-cbc -in file.txt -out file.enc

# 인증서 생성
openssl req -x509 -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365
```

**GnuPG (GPG)**

```bash
# 파일 암호화
gpg -c file.txt

# 공개키 암호화
gpg --encrypt --recipient email@example.com file.txt
```

**age (현대적 암호화 도구)**

- GitHub: https://github.com/FiloSottile/age
- 간단하고 안전한 파일 암호화

## 요약

**팁**

1. 이론과 실습을 병행
2. 공식 문서(NIST, OWASP)를 우선
3. 최신 보안 뉴스를 정기적으로 확인
4. CTF로 실전 감각을 키우기
5. 커뮤니티에 참여하여 질문하고 공유

**추가 업데이트** 이 문서는 2025년 1월 기준입니다. 암호화 기술은 빠르게 진화하므로:

- NIST 웹사이트에서 최신 표준 확인
- OWASP에서 최신 취약점 정보 확인
- CVE 데이터베이스에서 신규 취약점 모니터링

**질문이 있으면:**

- Stack Exchange Cryptography에 질문 올리기
- Reddit r/crypto에서 토론 참여
- GitHub Issues에서 라이브러리 개발자와 소통
