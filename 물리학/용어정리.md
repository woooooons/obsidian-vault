# **1. 원자(Atom)**

- 정의: 물질을 이루는 가장 작은 단위.
- 구성:
    - 원자핵(Nucleus): 양성자 + 중성자
    - 전자(Electron): 원자핵 주변을 도는 음전하 입자
- 중요 포인트:
    - 대부분 원자는 전자 수 = 양성자 수 → 전체 전하 중성
    - 원자핵은 매우 작지만 질량 대부분 차지

# **2. 원자핵(Nucleus)**

- 구성:
    - 양성자(Proton): + 전하, 스핀 때문에 자기 모멘트 존재(매우 작음)
    - 중성자(Neutron): 전하 0, 거의 자기적 성질 없음
- 역할: 원자의 중심, 질량 대부분 담당, 전자 끌어당겨 안정화

# **3. 전자(Electron)**

- 전하: –1, 항상 음전하
- 위치: 원자핵 주변 궤도
- 특징:
    - 스핀과 궤도운동으로 자기적 성질 발생
    - 전류를 만드는 실제 입자

# **4. 전하(Charge)**

- 정의: 전기적 힘을 만드는 성질
- 종류: + 전하(양성자), – 전하(전자)
- 특징: 질량 없음, 전자/양성자에 부착된 속성

# **5. 전류(Current)**

- 정의: 전하가 움직이는 흐름
- 실제: 전자 이동 = – → +
- 역사적 정의: 전류 방향 = + → – (프랭클린의 약속)
- 핵심: 실제 전자 이동과 전류 방향은 반대지만, 회로 해석은 약속된 방향 기준

# **6. 자기(Magnetic)**

- 정의: 움직이는 전하(전류)로 인해 발생하는 힘
- 특징:
    - N극과 S극 존재
    - 전류가 흐를 때만 자기장이 생김
    - 전자 스핀 때문에 원자 단위 자기장 가능

# **7. 전기(Electric)**

- 정의: 정지 또는 움직이는 전하가 만드는 힘
- 특징:
    - 정전기, 번개, 건전지 전류 등
    - 전하 사이에서 작용하는 힘

# **8. 분자(Molecule)**

- 정의: 여러 원자가 화학 결합으로 붙어 한 덩어리처럼 행동
- 예시: 물(H₂O) = 수소 2개 + 산소 1개
- 특징: 전자 공유로 전하 균형 유지, 극성 발생 가능

# **9. 관계도 한눈에 보기**

원자

├─ 원자핵

│   ├─ 양성자 (+ 전하)

│   └─ 중성자 (0 전하)

└─ 전자 (– 전하, 자기적 성질)

│

└─ 움직이면 전류 생성 → 자기 발생

전류(약속된 방향) = + → –

전자 실제 이동 = – → +

여러 원자가 결합 → 분자

전하와 전류 → 전기 현상

움직이는 전하 → 자기 현상

**핵심 포인트 요약**

1. 원자 = 원자핵 + 전자, 전자는 – 전하
2. 원자핵 = 양성자(+), 중성자(0)
3. 전류 = 전자가 실제로 이동하는 방향과 반대
4. 전기 = 전하가 만든 힘, 자기 = 움직이는 전하가 만든 힘
5. 분자 = 여러 원자가 결합한 단위, 전자 공유로 전하 균형
