## 1. 전류가 자기장을 만든다

### **물리 법칙**

- **앙페르 법칙**: ∇×**B** = μ₀**J**
    - 전류 밀도 **J**가 있으면 자기장 **B** 생성
- **비오-사바르 법칙**: d**B** = (μ₀/4π) · (I d**l** × **r̂**)/r²
    - 전류 I가 흐르는 도선 주변에 자기장 형성

### **실생활 예시**

- **솔레노이드(코일)**: 전류 흐르면 → N극, S극 형성 (전자석)
- **직선 도선**: 전류 주변에 동심원 자기장 (오른손 법칙)
- **전동기**: 코일에 전류 → 자기장 → 회전력 발생

### **핵심 포인트**

- 전류(움직이는 전하)는 **항상** 주변에 자기장 생성
- 전류 방향을 바꾸면 자기장 방향도 바뀜
- 전류가 클수록 자기장도 강함

## 2. 자기장이 전류를 만든다

### **패러데이 전자기 유도 법칙**

- **유도 기전력**: ε = -dΦ_B/dt
- **조건**: 자기선속 Φ_B가 **변해야** 함
- **∇×E = -∂B/∂t**: 변하는 자기장이 전기장 유도

### **전류가 유도되는 경우**

1. **자석을 코일에 넣거나 뺄 때**
    - 자기선속 변화 → 유도 전류 발생
2. **코일을 자기장 속에서 회전**
    - 발전기 원리 (화력/수력/풍력 발전)
3. **전자석 전류 변화**
    - 변압기 원리 (1차 코일 → 2차 코일)
4. **도체를 자기장 속에서 이동**
    - 로렌츠 힘: **F** = q(**v** × **B**)

### **중요: 정적 자기장은 전류를 만들지 않음**

- 자석을 코일 옆에 가만히 두면 → 전류 발생 안 함
- 반드시 **자기장의 변화** 또는 **상대 운동** 필요

## 3. 전자와 자기의 관계

### **(1) 거시적 관점 - 전류로서의 전자**

- **전자 이동(전류) → 자기장 생성**
- 전선, 회로, 전자석에서의 자기 현상
- 앙페르 법칙 적용

### **(2) 미시적 관점 - 원자 내 전자**

#### **① 전자의 궤도 운동**

- 전자가 원자핵 주위를 회전 → 원형 전류처럼 작용
- **궤도 자기 모멘트**: μ_L = -e·**L**/2m
- 각운동량 **L**에 비례하는 자기장 생성

#### **② 전자의 스핀**

- 전자는 고유한 각운동량(스핀 1/2) 보유
- **스핀 자기 모멘트**: μ_S = -g·e·**S**/2m (g ≈ 2)
- **스핀 자기 모멘트가 궤도보다 약 2배 강함**
- 이것이 **영구자석(Fe, Ni, Co)의 자성 근원**

### **(3) 물질의 자성 종류**

- **강자성(Ferromagnetic)**:
    - 전자 스핀이 한 방향으로 정렬 (Fe, Co, Ni)
    - 영구자석 가능
- **상자성(Paramagnetic)**:
    - 홀전자 존재, 외부 자기장에 약하게 끌림 (Al, O₂)
- **반자성(Diamagnetic)**:
    - 모든 전자가 쌍, 자기장에 약하게 반발 (Cu, Au, H₂O)

## 4. 최종 정리 - 전류와 자기의 상호작용

### **핵심 원리**

1. **전류 → 자기장**: 항상 성립 (앙페르 법칙)
2. **변하는 자기장 → 전류**: 조건부 성립 (패러데이 법칙)
3. **전자의 스핀/궤도 → 원자 자기장**: 영구자석의 원리

### 통합 이해 - 맥스웰 방정식

|법칙|수식|의미|
|---|---|---|
|가우스 법칙|∇·**E** = ρ/ε₀|전하 → 전기장|
|자기 가우스|∇·**B** = 0|자기 홀극 없음 (N-S 쌍)|
|패러데이|∇×**E** = -∂**B**/∂t|변하는 자기장 → 전기장|
|앙페르-맥스웰|∇×**B** = μ₀**J** + μ₀ε₀∂**E**/∂t|전류 + 변하는 전기장 → 자기장|

### 전자기파(빛)의 생성

- 변하는 전기장 → 변하는 자기장 → 변하는 전기장 → ...
- 이 순환이 **전자기파** 형성
- 빛, 전파, X선, 감마선 모두 동일 원리

## 5. 지구 자기장 - 특별한 케이스

### 지구는 거대한 자석

- **N극**: 캐나다 북부 (자기적 남극)
- **S극**: 남극 대륙 (자기적 북극)
- _방위자석 N극이 북쪽을 가리키는 이유_

### 지구 자기장의 원인 - 다이나모 이론

- 지구 외핵(액체 철-니켈)에서 **대류 전류** 발생
- 지구 자전 + 열대류 → 전류 형성
- 이 전류가 자기장 생성 (앙페르 법칙)
- **일반 자석과 다른 점**:
    - 영구자석(고체 스핀 정렬)이 아님
    - 액체 금속의 전류로 만들어짐
    - 시간에 따라 변함 (자극 역전 가능)
