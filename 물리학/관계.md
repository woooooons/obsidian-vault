# **1. 전류와 자기의 관계**

- 전류가 자기장을 만든다: 맞음
    - 움직이는 전하(전류)가 주변에 자기장을 형성
        - 예: 코일에 전류가 흐르면 자석처럼 행동 → 전자기 유도 원리
- 자기장이 전류를 만든다: 맞음, 하지만 조건이 있음
    - 자기장이 변하면 주변 도체에서 전류가 유도됨 → 패러데이 법칙
    - 정적인 자기장만으로는 전류가 생기지 않음

# **2. 전자와 자기의 관계**

- 전자는 스핀과 움직임 때문에 자기적 성질 있음 → 원자 단위에서 자기장 형성 가능
- 하지만 일반적으로 **‘자기 = 전류(움직이는 전자)’**라는 관점이 더 명확
- 따라서 “전자가 자기다”보다는
    - 전자 이동(전류) → 자기 발생
    - 전자 스핀 → 원자 자기장 발생 이렇게 구분하는 것이 정확

# **3. 정리**

1. 전류 → 자기장: 전자가 이동하면 주변에 자기장 생성
2. 변하는 자기장 → 전류 유도: 자기장이 변하면 도체에 전류 생김
3. 전자 자체가 자기장 생성: 스핀 때문에 가능, 하지만 원자 수준에서만 의미 있음
4. 결론적으로 “전류가 자기를 만들고 자기가 전류를 만든다”라는 문장은
    - 부분적으로 맞음: 조건을 명확히 해야 함
    - 전자는 직접 자기를 만들기보다 움직임과 스핀을 통해 자기장 생성

지구는 거대한 자석처럼 행동한다. 다만 이유와 방식은 일반 자석과 조금 다르다. 단계별로 정리해보자.
