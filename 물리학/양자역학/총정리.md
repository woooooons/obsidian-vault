# **양자역학 총정리**

# **1. 양자역학이란?**

- 정의: 원자와 미시 세계에서 물리량(에너지, 위치, 운동량 등)을 불연속적(quantized) 단위로 다루고, 확률로 기술하는 물리학 이론
- 목적: 고전역학으로 설명할 수 없는 현상(원자 스펙트럼, 광전효과, 흑체복사 등)을 설명

# **2. 핵심 개념**

**2-1. 양자화(Quantization)**

- 전자의 에너지, 원자 스펙트럼, 빛의 흡수/방출은 연속적이 아니라 특정 값만 가능
- 예:
    - 전자 궤도 에너지 = 불연속적
    - 광자 에너지 E = h\nu

**2-2. 파동-입자 이중성(Wave-Particle Duality)**

- 빛과 전자는 파동과 입자 특성을 동시에 가짐
- 예:
    - 파동: 간섭, 회절 현상
    - 입자: 광전 효과, 포톤 단위 흡수

**2-3. 불확정성 원리(Uncertainty Principle)**

- 위치(x)와 운동량(p)는 동시에 정확히 측정 불가
- 공식: \Delta x \cdot \Delta p \ge \frac{h}{4\pi}
- 의미: 미시세계에서는 확률로만 예측 가능

**2-4. 확률 해석**

- 전자의 위치, 속도는 확률 분포로 표현
- 예: 전자 구름(Electron cloud)
- 궤도는 ‘고정된 길’이 아니라 전자가 존재할 확률 영역

**2-5. 양자 얽힘(Quantum Entanglement)**

- 두 입자가 멀리 떨어져도 상태가 서로 연결되어 있음
- 하나의 상태를 측정하면 다른 입자의 상태가 즉시 결정

**2-6. 슈뢰딩거 방정식(Schrödinger Equation)**

- 전자의 파동함수 \psi를 통해 확률 밀도 계산 가능
- |\psi|^2 = 전자가 특정 위치에 있을 확률

# **3. 양자역학과 고전역학 차이**

|**구분**|**고전역학**|**양자역학**|
|---|---|---|
|상태|연속적|불연속적(quantized)|
|기술|위치, 속도 결정|확률로 기술|
|적용|거시적 물체|원자, 전자, 광자 등 미시세계|
|파동/입자|구분|파동-입자 이중성|
|에너지|연속|양자화됨|

# **4. 양자역학이 중요한 이유**

1. 미시세계 설명 가능: 원자, 분자, 전자, 빛
2. 현대 기술 기반: 반도체, 레이저, MRI, 양자컴퓨터
3. 고전물리 한계 극복: 광전효과, 원자 스펙트럼 등 설명

# **5. 한 줄 요약**

- “양자역학 = 미시세계의 에너지·운동을 불연속적 단위로 다루고, 확률과 파동-입자 이중성으로 설명하는 현대 물리학”
