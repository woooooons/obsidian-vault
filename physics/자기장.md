# **1. 지구 자기장(Magnetic Field of Earth)**

- 지구에는 **자기 북극(N)과 남극(S)**이 있음
- 나침반 바늘이 북쪽을 가리키는 이유가 바로 지구 자기장 때문

# **2. 지구가 자석처럼 행동하는 이유**

- 지구 내부 **외핵(액체 상태의 철·니켈)**이 빠르게 회전
- 외핵의 전도성 금속이 움직이며 전류를 생성
- 움직이는 전류 → 지구 자기장 생성
- 이 과정을 지오다이나모(geodynamo) 효과라고 부름

# **3. 일반 자석과의 차이**

|**구분**|**일반 자석**|**지구**|
|---|---|---|
|자기장 생성 원리|전자의 스핀과 원자 배열|액체 철·니켈의 대규모 전류(외핵)|
|자기장 방향|N극 → S극|자기 북극 → 자기 남극|
|변동|거의 일정|수천 년 단위로 변동 가능(극 이동)|

# **4. 핵심 요약**

- 지구는 거대한 자석처럼 행동
- 원인은 지구 내부 전류(액체 금속의 움직임)
- 나침반, 지구 자기장, 북극·남극 등 모두 지구 자성의 결과
